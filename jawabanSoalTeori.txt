1.	Pemrograman procedural adalah pemrograman yang Dimana kita mengatur data menggunakan function kalau di c++ kita menggunakan int main(){}
Pemrograman berorientasi objek adalah pemrograman yang Dimana kita mengatur data berbasis objek, jadi kita mengatur output dan input data menggunakan objek
2.	Konsep enkapsulasi adalah kita menyembunyikan atau menjadikan variabel/function tertentu hanya bisa diakses oleh class tersebut, kita menggunakan akses modifier protected atau private karena agar variabel/function yang berada didalamnya tidak bisa diakses oleh class yang lain, dan jika ingin mengakses cara mengaksesnya menggunakan setter dan getter
3.	Class adalah suatu wadah untuk menyimpan atribut atau method, sedangkan objek adalah suatu cara untuk mengakses atribut atau method tersebut, analogi dalam dunia nyata adalah class adalah jenis makhluk hidup seperti manusia sedangkan objek adalah kelakuannya misalkan: manusia tu biasanya ngapain aja, makan, minum dll
4.	Private, protected, public
5.	kegunaan constructor adalah untuk mempermudah kita dalam memasukkan atau mengatur data yang akan kita masukkan kedalam array, baik yang statis maupun yang dinamis, dan dia dieksekusi pertama kali Ketika program dijalankan
6.	function overloading membuat fungsi dengan nama sama tapi beda parameter
function overriding: ini adalah bagian dari polimorpi, jadi kita membuat satu function di base class kemudian function ini bisa diakses oleh derived class menggunakan pewarisan
7.	kegunaan dari destructor adalah untuk menghapus data yang sudah terpakai, dan dalam inheritance membutuhkan virtual destructor agar setelah program dijalankan data yang sudah terpakai bisa dihapus oleh si destructor ini dan tidak menyebabkan memory leaking atau datanya bocor sehingga mempengaruhi eksesusi program yang selanjutnya
8.	friend fuction dan friend class ini sama saja cara kerjanya dengan memberikan akses friend kepada derived class atau function yang akan kita jadikan dia friend function, dan sebaiknya kita menggunakan Ketika kita memerlukan beberapa/banyak class atau function yang isinya ada ingin kita gunakan pada function atau class tertentu, dan ini akan menghemat penggunaan memory dan waktu karena kita tidak perlu menuliskan kode lagi untuk membuat class atau function pada tingkatan derived class/function
9.	this kita gunakan Ketika kita menggukanan constructor atau function berparameter dan kita setting nama parameter tersebut sama dengan nama variabel yang kita deklarasikan di awal, kenapa kita menggunakan this? Karena untuk membedakan antara variabel global(variabel yang dideklarasikan diawal) dan nama parameter yang sudah kita setting di constructor atau function tersebut
10.	keyword static digunakan Ketika kita akan menambahkan value pada atribut sebuah class dengan static(yang valuenya sudah diset di awal bukan menggunakan user input), dan static juga bisa menjadikan method tersebut menjadi noOverriding atau hanya bisa diakses pada class tersebut, sama halnya dengan atribut yang di set static
